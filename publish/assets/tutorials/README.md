# デモシナリオ集

Insight AI Office のデモ・トレーニング用サンプルファイルです。
インストール直後にすぐ試せます。各シナリオ 1〜2 分で完結します。

## 基本シナリオ

| # | シナリオ | 入力 | 出力 |
|---|---------|------|------|
| 01 | 契約書 → 入出金 Excel | 下請負契約書 (.txt) | 入出金スケジュール (.xlsx) |
| 02 | 会議メモ → 議事録 | 会議メモ (.txt) | 議事録 (.docx) |
| 03 | 売上データ → 分析レポート | 月次売上 (.csv) | レポート (.docx) + 集計 (.xlsx) |
| 04 | 企画書 → プレゼン資料 | 企画書 (.txt) | プレゼン (.pptx) |
| 05 | 見積書テンプレート書き換え | テンプレート (.txt) | 見積書 (.docx) |
| 06 | 名刺画像 → 顧客リスト | 名刺写真 (.png) | 顧客リスト (.xlsx) |

## 部署別シナリオ

| # | 部署 | シナリオ | 入力 | 出力 |
|---|------|---------|------|------|
| 07 | 経理 | 請求書チェック・支払管理 | 請求書一覧 (.csv) | 支払管理表 (.xlsx) + 月次レポート (.docx) |
| 08 | 人事 | 採用選考の比較表 | 応募者情報 (.csv) | 選考比較表 (.xlsx) |
| 09 | 総務 | 社内通知文の作成 | プロンプトのみ | 社内通知 (.docx) |
| 10 | 営業 | 顧客向け提案書 | プロンプトのみ | 提案プレゼン (.pptx) |

## 使い方

1. Insight AI Office を起動
2. フォルダ内のファイルをチャット入力欄にドラッグ＆ドロップ
3. 同フォルダの「プロンプト例.txt」の内容をコピーしてチャットに貼り付け
4. 送信 → 成果物パネルに生成ファイルが表示されます

## テーマカラーのデモ

- シナリオ03: 「テーマはBlueで」→ 青色レポート
- シナリオ07: 「テーマはNavyで」→ 紺色の支払レポート
- シナリオ09: 「テーマはMonoで」→ モノクロ社内通知

## 動画撮影のポイント

1. 「添付 → 指示 → 生成 → 開く」の一連の流れ（30秒）
2. テーマカラーの切り替え（10秒）
3. 生成ファイルをSyncfusionエディタで開いて編集（20秒）
4. 「ChatGPTと違い、アプリ内で完結する」を強調
